Uploading Text

When uploading text, you have two options. But first of all, you go to /upload to upload text. Though, back to your options.


The first one is automatic, which is the default option. This option is the slowest, as I check each word. Additionally, this does not work if you want spanish phrases to be translated. Automatic only does it by each word.


The second one is custom chosen. With this mode you get to tell me what words are in spanish, which lets you seperate them by phrase/sentence if desired. To activate this mode, simply add !CUSTOM-CHOSEN as the first line when uploading. Then, put squigily brackets around each phrase. For example: {¿Cómo estás?}. When you click on it, it will say How are you? instead of como saying how, and estás saying to be.
